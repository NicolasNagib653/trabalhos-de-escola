Este botão que você usou foi uma demonstração de como mainupalar o DOM em javascript, utilizando a função onmouseenter / onmouseleave, a primeira função é acionada no momento que alguém passar o mouse por cima do objeto, já a segunda é sempre que o mouse sair do objeto.

Uma outra função utilizada foi o onclick que é acionado sempre que alguém clica no objeto
